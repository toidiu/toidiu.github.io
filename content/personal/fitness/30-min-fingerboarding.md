30m MacLeod Hangboard Routine
## WARMUP
### Jug
- 10s hang
  - 1m rest
- 10s hang
    - 1m rest
- 5-6 pullups
    - 1m rest
- 5-6 pullups
    - 1m rest

### 21mm
- 10s chisel
    - 1m rest
- 10s half crimp
    - 1m rest
- 10s 3-finger-drag
    - 1m rest

### 14mm
- 10s half crimp
    - 2m rest

## WORKING SETS: Weighted to failure @ 10 seconds
### set 1
- 10s chisel
    - 2m rest
- 10s chisel
    - 2m rest
- 10s chisel
    - 2m rest

### set 2
- 10s half crimp
    - 2m rest
- 10s half crimp
    - 2m rest
- 10s half crimp
    - 2m rest

### set 3
- 10s 3-finger drag
    - 2m rest
- 10s 3-finger drag
    - 2m rest
- 10s 3-finger drag
    - 1m rest

## Jug
- 5-6 pullups
